# Math Thinking RPG

這是一個純靜態的數學思考 RPG 網站。學生輸入玩家姓名後進入復古 2D 大廳，與 NPC 對話進入四個挑戰地圖。

## 關卡

- 家事排程塔：用甘特圖安排自動與手動任務。
- 沙漏時計洞：操作 7 分鐘與 10 分鐘沙漏，量測 25 分鐘。
- 巴士調度營：比較大客車與中型巴士的成本與座位。
- 警衛雅克之門：用字母轉數字工具推理遺失鑰匙標籤。

## 進度

玩家姓名、完成關卡、嘗試紀錄與通關狀態會儲存在瀏覽器 `localStorage`。畫面提供「清空進度、重新開始」按鈕，可以完整重置。

## 使用方式

直接開啟 `index.html` 即可遊玩，不需要伺服器、資料庫或外部 API。
